# Superset Dashboard Competencies Assessment Checklist

> Self-assessment checklist for tracking progress toward committer-level competencies

## Instructions

Use this checklist to assess your current competency level and identify areas for growth. Rate each skill:
- `[ ]` = Not yet learned
- `[~]` = Basic understanding
- `[x]` = Proficient

---

## Level 1: Contributor

### Frontend Fundamentals
- [ ] Can create functional React components with hooks
- [ ] Understand TypeScript type annotations and interfaces
- [ ] Know how to use `useState`, `useEffect`, `useCallback`
- [ ] Can write Jest unit tests for components
- [ ] Familiar with ESLint and Prettier configuration

### Backend Fundamentals
- [ ] Comfortable with Python 3.9+ syntax
- [ ] Understand Flask request/response cycle
- [ ] Can write pytest unit tests
- [ ] Know how to use type hints in Python
- [ ] Familiar with Black and isort formatters

### Git & Workflow
- [ ] Can fork and clone repositories
- [ ] Know how to create feature branches
- [ ] Can write clear commit messages
- [ ] Understand PR process and templates
- [ ] Can respond to code review feedback

### Development Environment
- [ ] Can set up local Superset dev environment
- [ ] Know how to run frontend dev server
- [ ] Can run backend with hot reload
- [ ] Understand Docker Compose basics
- [ ] Can run test suites locally
